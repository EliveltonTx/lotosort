📲 Como instalar no celular
Android (Chrome):
1. Abra o link no Chrome
2. Toque no menu ⋮ (canto superior direito)
3. Toque em "Adicionar à tela inicial"
4. Confirme → ícone aparece como um app! 🎉
iPhone (Safari):
1. Abra o link no Safari
2. Toque no ícone de compartilhar (□↑)
3. Role e toque em "Adicionar à Tela de Início"
4. Confirme → ícone na tela inicial! 🎉
Hospedar grátis no GitHub Pages:
1. Crie conta em github.com
2. Crie um repositório público chamado lotosort
3. Faça upload deste arquivo index.html
4. Vá em Settings → Pages → Branch: main → Save
5. Seu link será: seuusuario.github.io/lotosort
